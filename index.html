Hướng dẫn cài DeskIn trên macOS

Xem các bước Tải DeskIn

Giới thiệu nhanh

DeskIn là phần mềm điều khiển máy tính từ xa. Trên macOS, ngoài việc cài ứng dụng, bạn còn cần cấp một số quyền hệ thống như Accessibility và Screen Recording để điều khiển máy đầy đủ.

  • Tải đúng bản dành cho macOS.
  • Mở file cài đặt và thêm ứng dụng vào Applications.
  • Cấp quyền trong System Settings.
  • Đăng nhập và kết nối bằng ID thiết bị.

Các bước cài đặt

Bước 1

Tải DeskIn cho macOS

Vào trang tải chính thức của DeskIn và chọn bản dành cho Mac để đảm bảo đúng phiên bản mới.

  • Mở website DeskIn.
  • Chọn bản macOS.
  • Chờ file tải xong.
Bước 2

Mở file cài đặt .dmg

Sau khi tải xong, mở Finder và vào thư mục Downloads để chạy file cài DeskIn.

  • Mở Finder.
  • Vào Downloads.
  • Nhấp đúp vào file DMG.
Bước 3

Kéo DeskIn vào Applications

Đây là cách cài phổ biến trên macOS. Sau khi sao chép xong, bạn có thể mở DeskIn từ Applications hoặc Launchpad.

  • Kéo biểu tượng DeskIn.
  • Thả vào thư mục Applications.
  • Mở ứng dụng lần đầu.
Bước 4

Cấp quyền cần thiết

Để điều khiển từ xa đầy đủ, bạn cần bật các quyền mà DeskIn yêu cầu trong phần bảo mật của macOS.

  • Mở System Settings.
  • Vào Privacy & Security.
  • Bật Accessibility, Screen Recording và khi cần là Full Disk Access.
Bước 5

Đăng nhập và kết nối

Sau khi cài xong, mở DeskIn, đăng nhập và dùng ID của thiết bị để kết nối từ xa.

  • Đăng nhập hoặc tạo tài khoản.
  • Ghi lại ID và mật khẩu.
  • Nhập ID máy cần kết nối.

Lưu ý khi sử dụng

  • Nếu chỉ xem được màn hình mà không điều khiển được, hãy kiểm tra lại quyền Accessibility và Screen Recording.
  • Giữ nguyên file index.html và thư mục images khi đưa lên GitHub Pages.
  • Chỉ chia sẻ ID và mật khẩu cho người bạn tin tưởng.
